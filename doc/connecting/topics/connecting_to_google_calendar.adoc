= Connecting to Google Calendar
:context: calendar

To trigger execution of an integration by getting events from a 
particular Google Calendar, add a Google Calendar connection to an integration as its
start connection. In an integration, to obtain, update, or create a Google
Calendar event, add a Google Calendar connection as the integration's
finish connection, or as a middle connection. 

Details for connecting to Google Calendar are in the following topics:

* <<register-with-google-calendar_{context}>>
* <<enable-google-calendar-api_{context}>>
* <<create-google-calendar-connection_{context}>>
* <<add-google-calendar-connection-start_{context}>>
* <<add-google-calendar-connection-finish-middle_{context}>>
* <<add-google-calendar-connection-to-get-one-event_{context}>>
